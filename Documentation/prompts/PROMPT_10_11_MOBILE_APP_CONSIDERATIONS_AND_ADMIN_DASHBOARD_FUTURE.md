## PROMPT 10: MOBILE APP CONSIDERATIONS (Future)

If creating mobile apps (iOS/Android):

### FRAMEWORK OPTIONS:
• React Native (use existing React code)
• Flutter (better performance)
• PWA (web app that installs like native)

### MOBILE-SPECIFIC FEATURES:
• Camera integration (take gem photo directly)
• Offline mode (save drafts locally)
• Push notifications (design ready, new features)
• Share to social media natively
• AR preview (view jewelry on hand - advanced)

### PWA REQUIREMENTS:
• Service worker for offline
• manifest.json file
• Install prompt
• App icons (various sizes)
