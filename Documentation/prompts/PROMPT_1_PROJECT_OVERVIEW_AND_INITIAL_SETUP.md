## PROMPT 1: PROJECT OVERVIEW & INITIAL SETUP

### Create the Application
Create a web application called "AI Jewelry Designer" that allows users to design custom jewelry using AI image generation.

### PROJECT OVERVIEW
- Users can upload images of their gemstones
- Users fill out a structured form about their gem (type, cut, size, color, transparency)
- Users write a creative prompt describing the jewelry they want
- The app uses Google Gemini 2.5 Flash Image API to generate 3-4 concept designs
- Users can select a favorite design and refine it with conversational prompts
- Users can save and download their designs

### TECH STACK
- **Frontend:** React with modern UI components
- **Styling:** Tailwind CSS
- **AI API:** Google Gemini 2.5 Flash Image API
- **Storage:** Cloud storage for uploads and generated images
- **Authentication:** User accounts to save designs
- **Database:** For storing user designs and refinement history

### IMPORTANT REQUIREMENTS
- Mobile-responsive design
- Clean, professional jewelry/luxury brand aesthetic
- Fast loading times
- Clear error handling
- Security best practices for API key management
